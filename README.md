## 关于我们 ♪
小牛哞哞，一个听起来就让人忍俊不禁的名字，源于主理人家中一个属牛的小朋友。
这个名字承载着我们对音乐最初的心意——
纯粹、热爱、不拘一格。

我们相信：
音乐不应该被框死在规则里，
创作不该被限定在某个风格中。
就像一头倔强的小牛，
带着坚定，
带着温暖，
带着一点点可爱，
去创造属于每个作品的独特声音。
